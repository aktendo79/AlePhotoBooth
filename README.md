# AlePhotoBooth
Here you can find all the working scripts Project and the features of my PhotoBooth

- <b>Raspberry Pi</b>
  - RaspbianOS (flashing microsd and first configuration of the OS)
- <b>PiBooth</b>
  - Installation
    - Configuration of PiCamera settings
    - Configuration of External Hard Disk settings to save images there
    - Configuration of external plugins (Neopixel)
  - QRcode plugin installation and configuration
    - The plugin generates a QRcode to download the corresponding image
  - HTML gallery page where you can see all the photos taken and download them
    - Hard disk local + Cloudflare tunnel + GoDaddy domain
    - Every photos shows its QRcode below and at the very bottom a there is a QRcode to download all the photos taken
  - Random gallery page
    - When PiBooth is in idle mode it opens a page with the photos taken in random order
    - When a button is pressed it closes the page and PiBooth is shown again
- <b>Neopixel ring 24 leds</b>
  - Python script
  - RaspbianOS configuration
  - PiBooth configuration
- <b>Thermal Printer</b>
  - Python script
  - RaspbianOS configuration
  - PiBooth configuration
- <b>Arcade Led Buttons</b>
  - Python script
  - RaspbianOS configuration
  - PiBooth configuration
